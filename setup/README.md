# Setup new Desktop

## Common Steps (Required)

### Clone the repository

```sh
git clone git@github.com:bratushkadan/dotfiles.git $HOME/dotfiles && cd $HOME/dotfiles
```

### Install fonts

Run `./fonts/install` from the *dotfiles* repo root.
